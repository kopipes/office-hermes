# Provaliant Brain OS - User Guide

## Entry Commands
- `/entry project_update`
- `/entry meeting`
- `/entry vendor_quote`
- `/entry budget`

## Query Commands
- `/status <project>`
- `/vendor compare <item>`
- `/budget <project>`
- `/report weekly`

## Response Contract
Every response should include:
- answer
- source
- last updated
- confidence
- next suggested action
