# Skill: weekly_project_report

## Purpose
Generate weekly management report grouped by business unit.

## Data Sources
- projects
- project_updates
- budgets
- action_items
- meetings
- decisions
- gbrain relationships

## Output Format
1. Executive Summary
2. Projects On Track
3. Projects At Risk
4. Budget / Margin Issues
5. Open Decisions
6. Open Action Items
7. Required Chandra Attention
8. Next Week Priorities

## Rules
- SQL first, then wiki/evidence/context
- attach source references
- concise and action-oriented
